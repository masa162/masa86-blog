## Supported Sitemap Formats

Google Search Console supports three primary sitemap formats: XML, RSS/Atom, or text. More specifically:

- **XML sitemaps**: Standard format with full metadata support
- **Sitemap index files**: A sitemap of sitemaps
- **RSS feeds**: Sitemap in RSS feed format
- **Atom feeds**: Sitemap in Atom feed format
- **Text sitemaps**: A simple list of URLs in a text file that must be UTF-8 encoded, and cannot be more than 50MiB (uncompressed) or contain more than 50,000 URLs

## XML Sitemap Specifications

### File Requirements

The sitemap file must be UTF-8 encoded, and must observe size limits of 50MB/50,000 URLs.

### XML Structure

Sitemaps must begin with an opening <urlset> tag and end with a closing </urlset> tag, and specify the namespace (protocol standard) within the <urlset> tag.

### Date Format

Dates must use W3C Datetime encoding, although you can omit the time portion, but if you do specify a time, you must also specify a time zone.

### Special Characters

All data values in a Sitemap must be entity-escaped, and as with all XML files, any data values (including URLs) must use entity escape codes.
